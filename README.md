# CodeJudge Transactions, Safe Changes & DB Reliability

## Objective

This project demonstrates safe database modification operations and transaction reliability concepts using the CodeJudge database.

The project covers:

- Safe UPDATE operations
- Safe DELETE operations
- Transactions using COMMIT and ROLLBACK
- SAVEPOINT usage
- ACID property explanation
- Reliability incident analysis
